# Upcode Open Source

Welcome to **Upcode OSS** – the open source ecosystem of **Upcode**.

This organization contains public projects, tools, and infrastructure components developed and maintained by Upcode. Our goal is to build reliable, developer-friendly software that helps teams work with modern infrastructure, automation, and scalable systems.

---

## What We Build

We focus on tools that support modern development and infrastructure:

- 🖥️ **Server & Infrastructure Management**
- ⚙️ **Automation & Developer Tooling**
- 🧩 **Self-Hosted Platforms**
- 🔐 **Security-focused Utilities**
- 🌐 **Web & API Development**

Our projects are built with a **Linux-first and open-source-friendly mindset**.

---

## Philosophy

At Upcode we believe in:

- **Open standards**
- **Automation over manual work**
- **Security by default**
- **Clear and maintainable code**
- **Developer-first tooling**

We build software that is designed to be **simple, reliable, and scalable**.
